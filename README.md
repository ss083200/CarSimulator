## PreRequisites

- g++ --version
- sudo apt-get install libsfml-dev
- sudo apt install cmake
- sudo apt-get install libgtest-dev
- cd /usr/src/gtest
- sudo cmake CMakeLists.txt
- sudo make
- sudo cp lib/*.a /usr/lib

## Go TO Downloads/
- git clone <repo>
- rm -rf build/
- mkdir build
- cd build/
- cmake ..
- make

## To Run The application 

- ./CarProject

## To Run the tests

- ./CarTests
